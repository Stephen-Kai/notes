## 两数之和

不管是两数之和还是三数之和, 都是可以使用 map 结构来解决的一个问题

1. 暴力 O(n^2) x + y = ?
2. Set (O(n)) x + y = 9 => y = 9 - x, 使用 Set

## 三数之和

1. 暴力 a, b, c => 3 Loops O(n^3)
2. c = -(a + b) O(n^2)
3. sort array, 把数组重新排了份序, 一头一尾两个指针, a 为数组第一个元素, 如果 a + b + c > 0, right --, 如果 a + b + c = 0, left ++, 两端不断地往中间夹, 逼进目标值

推荐第二个做法
