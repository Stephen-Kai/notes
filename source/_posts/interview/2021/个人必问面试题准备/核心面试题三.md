### 实现 lodash 的 get 函数

百度一面的题目, 说是很简单的题

#### 考察点

1. 基础方法的运用能力
2. 很简单很基础的编程题, 突然很想去把 lodash 的源码刷了

```js
// 使用:
// get(data, 'a[3].b')
// get(data, 'a.b.c')

// 实现:
const get = (data, path, defaultValue = void 0) => {
  // a[1].b -> a.1.b, 使用捕获分组
  const paths = path.replace(/\[(\d+)\]/g, ".$1").split(".");

  // paths -> ['a', '1', 'b']
  let result = data;
  for (const path of paths) {
    result = Object(data)[path];
    if (result === null) {
      return defaultValue;
    }
  }

  return result;
};
```
